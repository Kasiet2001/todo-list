## Описание

Это базовое ToDo-приложение с возможностью создавать, получать, обновлять и удалять задачи. Используется FastAPI для быстрого и удобного создания API, а SQLite — для хранения данных.

---

## Технологии

- Python 3.10+
- FastAPI
- SQLAlchemy
- SQLite
- Uvicorn
